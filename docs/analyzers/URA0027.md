# URA0027: Udon does not support method parameters of this type

Udon does not support method parameters of this type

## Examples

```csharp
using System;

using UdonSharp;

class ExampleClass : UdonSharpBehaviour
{
    private void Update(IntPtr ptr) // URA0027
    {
    }
}
```

## Solution

```csharp
using UdonSharp;

class ExampleClass : UdonSharpBehaviour
{
    private void Update()
    {
    }
}
```
