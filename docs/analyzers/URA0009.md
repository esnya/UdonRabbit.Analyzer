# URA0009: Base type calling is not yet supported by U#

Base type calling is not yet supported by UdonSharp

## Examples

```csharp
using UdonSharp;

class ExampleClass : UdonSharpBehaviour
{
    private void Update()
    {
        base.GetComponent<Transform>(); // URA0009
    }
}
```

## Solution

```csharp
using UdonSharp;

class ExampleClass : UdonSharpBehaviour
{
    private void Update()
    {
        GetComponent<Transform>();
    }
}
```
