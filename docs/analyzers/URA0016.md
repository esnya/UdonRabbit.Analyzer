# URA0016: Udon does not support the 'Awake' event, use 'Start' instead

Udon does not support the 'Awake' event, use 'Start' instead

## Examples

```csharp
using UdonSharp;

class ExampleClass : UdonSharpBehaviour
{
    private void Awake() // URA0016
    {
    }
}
```

## Solution

```csharp
using UdonSharp;

class ExampleClass : UdonSharpBehaviour
{
    private void Start()
    {
    }
}
```
